# Smart Queue & Crowd Predictor

A campus-focused solution to reduce waiting time by predicting crowd density in shared spaces like canteens and libraries using Google technologies.

## Problem
Students waste time standing in long queues without knowing peak hours.

## Solution
A real-time crowd monitoring and prediction system using QR check-ins and Google Firebase.

## Google Technologies Used
- Firebase Firestore
- Firebase Cloud Functions
- Google Maps API
- Vertex AI (future scope)

## Features
- Live crowd count
- Peak hour prediction
- QR-based entry/exit
- Scalable for any campus

## Impact
- Reduced waiting time
- Better space utilization
- Improved student experience
